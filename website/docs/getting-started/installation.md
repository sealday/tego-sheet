# Installation

tego-sheet requires Node.js 20.19.0 or newer.

Install the package from npm:

```bash
npm install tego-sheet
```

Import the package stylesheet once in your application entry point:

```tsx
import 'tego-sheet/styles.css';
```

React and React DOM are peer dependencies. The current release expects `react` and `react-dom`
compatible with version 19.2.7.
